*for comando usado para realizar un bucle dentro de la estructura de un programa para hacer una accion repetitiva, se deben definir una variable con inicio y fin.
/////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
&&, es un comando logico el cual devuelve true solo si las dos condiciones se cumplen o false si una de ellas no se cumple.
/////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
||, es un comando logico el cual devuelve true solo si cumple con una de dos condiciones o false si no cumple ninguna condicion
/////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
!, operador logico que sirve para invertir el valor a una variable, es decir a un boleano true devolvera false y aun boleano false devolvera true.
